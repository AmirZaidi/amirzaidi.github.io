Logische Uitdrukkingen

We beginnen bij het begin, de logische uitdrukkingen. Dat klinkt als een moeilijke term, maar eigenlijk is het nog makkelijker dan een formule in wiskunde: 

VariabelType NaamVanVariabel = WaardeVanVariabel;
string AchternaamVanRens = "Kievit";
char LaatsteLetter = 't';
int LeeftijdVanRens = 15;
bool RensDoetIets = false;

De eerste regel is een voorbeeld van hoe logische uitdrukkingen eruit zien. Daarna geef ik je vier voorbeelden. De naam van een variabel mag letters, cijfers en lage streepjes bevatten, maar het mag niet met een cijfer beginnen. De semicolon ';' betekent het einde van een logische uitdrukking. Het beste is dat je altijd na de semicolon een enter gebruikt. Je hebt heel veel variabeltypen en je kan zelfs nieuwe maken, maar hier houden we het even op de vier simpelste:

  • String: Een tekst -> Waarde moet tussen dubbele apostrof
  • Char: Één letter -> Moet tussen enkele apostrof, kan niet leeg zijn
  • Int: Een heel nummer van -2 tot 2 miljard (32 bits) -> Waarde los
  • Bool: Een waar of niet waar (true/false) -> Waarde als woord

Je hebt nog veel meer veelgebruikte variabeltypen zoals de uint (0 tot 4 miljard), byte (0 tot 255) en double (kommagetal). Getallen hoeven nooit tussen apostrofs.

Zodra je een variabel hebt gemaakt en je hem wilt veranderen hoef je niet nog een keer te zeggen welk soort variabel het is. Dat mag zelfs niet, en je kan het variabel type niet veranderen in C# nadat je een naam een variabel type hebt gegeven.

Rekenen met variabelen

Variabelen zijn bedoeld om in een berekening te gebruiken, en die berekeningen lijken gewoon op simpel rekenen: 

int GeboorteJaar = 2006;
int Leeftijd = 3;
int HetJaarNu = GeboorteJaar + Leeftijd;

Of een simpelere optelling: 

int HetJaarNu = 2006 + 3;

Of met een begingetal en een vast optelgetal: 

int GeboorteJaar = 2006;
int HetJaarNu = Geboortejaar + 3;

Daarna kan je die variabel bijvoorbeeld opslaan in een grote database of gebruiken in de volgende berekening. Teksten bij elkaar voegen is ook mogelijk met het plus teken: 

string Voornaam = "Rens";
string Achternaam = "Kievit";
string Helenaam = Voornaam + " " + Achternaam;

De variabel Helenaam is nu dus Rens Kievit (Als je dacht dat de apostrof erbij hoorde, de apostrof laat alleen zien dat het een tekst is, het zit er niet letterlijk in)
Berekeningen met nummers zijn ook heel makkelijk in te korten, je kan bijvoorbeeld als je een variabel recyclet: 

int JaarNu = 2003;
int TijdVerstreken = 5;
JaarNu = JaarNu + TijdVerstreken;

Ook bijvoorbeeld dit doen: 

int JaarNu = 2003;
int TijdVerstreken = 5;
JaarNu += TijdVerstreken;

En er is zelfs een speciaal teken voor het nummer 1 erbij optellen of eraf halen, deze twee dingen betekenen hetzelfde (behalve dat de variabelen anders heten): 

int JaarNu1 = 2009;
Jaarnu1 += 1;
Jaarnu1 += 1;
Jaarnu1 -= 1;

int JaarNu2 = 2009;
Jaarnu2++;
Jaarnu2++;
Jaarnu2--;

Jaarnu1 en Jaarnu2 zijn nu allebei 2010.

Welke zijn fout?

  1. char EersteLetter = "a";
  2. string Tekst = "Hallo"
  3. int EenNummer = 5.8;
  4. bool ProgrammaKlaar = true;
  5. char 5deLetter = 'e';

Uitwerking

    Ze zijn allemaal fout behalve de 4e:
  1. Gebruikt de dubbele apostrof voor een char
  2. Vergeet de semicolon
  3. Gebruikt een int variabel om een kommagetal op te slaan
  4. Variabelnaam begint met een cijfer

C# Syntax

De syntax in C# is niet heel ingewikkeld vergeleken met bijvoorbeeld C++. C# gebruikt woorden, is-tekens, punten, komma's, twee soorten apostrof, drie soorten haakjes en de semicolon het meest. Waarvoor het allemaal is zal je dit hoofdstuk leren.

De Semicolon ';'

De semicolon geeft het einde van een logische uitdrukking aan. Het is altijd de bedoeling dat je een enter na een semicolon gebruikt, met uitzondering van de for en do while-construct. Na bijvoorbeeld een variabel is gelijk aan een waarde gebruik je hem, maar ook na het gebruiken van een functie. Bij vragen met if/else en constructies zoals foreach of while gebruik je hem niet.

Conversies

Meestal krijg je als invoer van een gebruiker een string, maar wat nou als het een nummer hoort te zijn en je ermee wilt rekenen? Bijvoorbeeld als iemand zijn verjaardag opgeeft als nummers en je wilt bekijken of de dag vandaag hetzelfde is als die dag. Maar je kan gewoonlijk strings en ints niet met elkaar vergelijken. Dan moet je je string converteren naar een int. Dat is simpel te doen, maar als het niet omgezet kan worden zal je een fatale error krijgen en je programma crashen. Later in het hoofdstuk leer je hoe je dat kan oplossen. 

string Dag = "6";
string Maand = "9";
int DagNummer = (int)Dag;
int MaandNummer = (int)Maand;

Je zet gewoon simpelweg het type waarheen je het wilt converteren ervoor tussen ( ). Helaas zal dit voorbeeld om logische redenen vastlopen (September kan niet worden omgezet naar een getal): 

string Verjaardag = "6 September";
int Verjaardag = (int)Verjaardag;
