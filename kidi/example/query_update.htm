Het gebruik van een UPDATE query

Database Tabel 'artikelen':
 id	| titel				| verhaal
 1	| Welkom op Kidi!	| BlaBla1
 2	| Nieuwe functies	| BlaBla2
 3	| Bijna afgerond		| BlaBla3
Update.php:
<?php
$db = new mysqli('localhost', 'root', '******', 'kidi_website');

$id = intval($_GET['id']);
$titel = $db->real_escape_string($_GET['titel']);

$db->query("UPDATE artikelen SET titel = '" . str_replace('_', ' ', $titel) . "' WHERE id = '" . $id . "'");
?>
