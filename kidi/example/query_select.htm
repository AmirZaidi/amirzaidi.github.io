Het gebruik van een SELECT query

Database Tabel 'artikelen':
 id	| titel				| verhaal
 1	| Welkom op Kidi!	| BlaBla1
 2	| Nieuwe functies	| BlaBla2
 3	| Bijna afgerond		| BlaBla3
Nieuws.php:
<?php
$db = new mysqli('localhost', 'root', '******', 'kidi_website');
$nieuws = $db->query('SELECT * FROM artikelen LIMIT 2');
while ($artikel = $nieuws->fetch_assoc())
{
	echo '<div><b>', $artikel['titel'], '</b></div>', "\n";
	echo '<div>', $artikel['verhaal'], '</div>', "\n";
	echo '<div><a href="http://', $artikel['id'], '.com">Lees Meer</a></div>', "\n\n";
}
?>				
Uitkomst:
<div><b>Welkom op Kidi!</b></div>
<div>BlaBla1</div>
<div><a href="http://1.com"></a></div>

<div><b>Nieuwe functies</b></div>
<div>BlaBla2</div>
<div><a href="http://2.com"></a></div>

Fetch_assoc is een functie van de class mysqli_result (waarvan je een object krijgt als resultaat wanneer je de functie $db->query() gebruikt met een SELECT query), die de volgende rij waardes zet in een array. Zodra er geen waardes meer over zijn wordt fetch_assoc() dus FALSE en stopt de while loop.
