HTML

HTML is een computertaal die wordt gebruikt voor het maken van websites. In HTML werkt men met zogenaamde tags, die aangeven hoe bepaalde stukken geschreven tekst moeten worden aangegeven door de browser die de HTML code omzet in een website.

Beginnen met HTML

Om te beginnen met het maken van een website. Moet je eerst aangeven dat je gaat coderen in HTML. 

<!DOCTYPE html>
   <head>

   </head>
   <body>

   </body>
</html>

De html tag geeft hier aan dat het document een html bestand is, en het !DOCTYPE gedeelte geeft aan dat je aan het programmeren bent in HTML5.
In de head tag zet je alle informatie over de website, deze krijg je meestal niet direct te zien op de pagina zelf. Wel kun je hier de titel inzetten, die je helemaal bovenaan in de browser ziet.
Tussen de body tags komen alle dingen die je uiteindelijk op de website ziet. Dit is waar het grootste gedeelte van de website in komt.

Tekst

Om tekst in te voeren op de website, zou je gewoon tekst kunnen invoeren in je kladblok bestand. Maar dit is niet volgens de regel, en het is moeilijk om aan te passen. Ook voor tekst hebben we verschillende tags. Dit zijn de <p> en de <h*> tags. 

<!DOCTYPE html>
   <head>
   </head>
   <body>
      <h1>Programmeren</h1>
      <p>Programmeren is heel leuk!</p>
   </body>
</html>

In het voorbeeld hieboven staat er nu iets in de body, het stuk dat op de uiteindelijke website komt. Hierin staan de twee verschillende tags waar ik het net over had. De h-tag geeft een heading aan, de tekst hiertussen, nu "Programmeren".
Deze tekst wordt groter, en dikgedrukt weergegeven. Dit wordt meestal gebruikt voor titels. Van de h-tag zijn verschillende varianten. h1-h6. Hoe groter het nummer achter de h, hoe kleiner de tekst ertussen wordt.
De p-tag geeft een paragraaf aan. Hierin zet je meestal het grootste stuk van de tekst. De meeste browser zetten voor en na het begin van een paragraaf wat ruimte.

Tags gebruiken

Zoals ik net zei, gebruik je in HTML heel erg veel tags, maar tags kan je niet zomaar opschrijven. Om een tag aan te geven moet je het tussen <> zetten. Bijvoorbeeld <html> of <body>.
Als je een tag hebt geopend moet je deze daarna ook afsluiten. Dit doe je door een / in de tag te zetten. </html> en </body>

Code invoeren

Als je jouw HTML code aan het invoeren bent maakt het niet uit of je de tags in kleine letters, of in hoofdletters zet. Ook maken extra spaties in de code niet uit.
Wel kan ik je aanraden om in tags kleine letters te gebruiken, omdat het sneller is. En als je de code aan het typen bent, meer wit ruimte te maken als je tags nest.

Nesten

Nesten is iets belangrijks in HTML. Als je verschillende tags in elkaar gebruikt. Moet je de laatst geopende tag, als eerst sluiten. Dan is het dus: 

<!DOCTYPE html>
   <head>

   </head>
   <body>

   </body>
</html>

En niet: 

<!DOCTYPE html>
   <head>

   <body>

   </head>
   </body>
</html>
