Attributen

In html wordt naast tags, ook veel gebruik gemaakt van attributen. Attributen zijn extra stukjes inforamtie die achter een tag komen. Attributen komen altijd meteen achter de tag, tussen de <>. Na de tag schrijf je een ="".
Tussen de aanhalingstekens komt een waarde van het attribuut, de "value".

Een paar van deze attributen zijn al gegeven in de vorige paragrafen. Bij de img-tag hoort bijvoorbeeld het attribuut src. De value van dit attribuut is een link naar een afbeelding. Src geeft hier aan dat de value erachter een link is naar een afbeelding.

Stylen

Attributen worden meestal gebruikt voor het stylen van bepaald stukken informatie op een website. Hiervoor is het attribuut style. Dit attribuut heeft veel verschillende values. Hiernaast is bijvoorbeeld aangegeven hoe je de kleur van een paragraaf kan veranderen. Op diezelfde manier kan je het lettertype,
font-family: , of de groote van de letters aanpassen, font-size: .

Global Attributes

…Èn van de globale attributen staat hierboven. Het attribuut style kan bij iedere tag gebruikt worden. Er bestaan heel veel verschillende values bij style, maar deze komen aan de orde in het volgende hoofdstuk.
Ook de attributen class en id maken beide deel uit van CSS, en leg ik ook in het volgende hoofdstuk uit.

Andere dingen die je met de globle attributen kan doen, zijn bijvoorbeeld de bezoekers een stuk tekst laten veranderen, contenteditable="true|false" aangeven of een afbeelding of een stuk tekst wel of niet gesleept mag worden, draggable="true|false" of de uitlijning van een paragraaf aangeven. dir="ltr|rtl|auto"

Values

Soms bestaat een value uit meer dan alleen ÈÈn stuk. Bijvoorbeeld bij color. Color is een value van het attribuut style. Maar, als je alleen aangeeft dat het stuk teskt een kleur moet hebben, weet de browser niet welke kleur hij het stuk tekst moet geven. Dit doe je door de kleur te geven achter een dubble punt. Als je een rode paragraaf wilt hebben, ziet dat er zo uit 

<p style="color:red">Rode tekst</p>
